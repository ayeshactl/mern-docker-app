# 🚀 Dockerized MERN Stack Todo App

A full-stack Todo application built using MERN stack and deployed using modern tools.

---

## 🌐 Live Demo

👉 https://todo-app-one-nu-97.vercel.app  

---

## 🛠 Tech Stack

- React.js (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)
- Docker (Containerization)

---

## ⚙️ Features

- Add Todo
- Delete Todo
- Toggle Complete
- REST API Integration

---
